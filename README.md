## 注册 get

- 发送

  ```
  http://47.110.144.145:4567/register?name=jack&pwd=123456
  ```

- 返回

  成功

  ```
  {
      "register": "0"
      "token": "xxxxxxxxxxxxx"
  }
  ```

  失败

  ```
  {
  	"register": "-1"
  	"error": "0,1,2,3"
  }
  ```

  0：用户名已存在

  1：用户名不符合规范

  2：密码不符合规范

  3：其他

## 登录 get

- 发送

  ```
  http://47.110.144.145:4567/login?name=jack&pwd=123456
  ```

- 返回

  成功

  ```
  {
      "login": "0"
  }
  ```

  失败

  ```
  {
      "login": "-1"
      "error": "0,1,2"
  }
  ```

  0：用户名不存在

  1：密码不匹配

  2：其他
